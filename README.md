# Sistema de Análisis de Finanzas Personales V2

## Visión general

Sistema de finanzas personales basado en reglas explícitas, orientado al control real de liquidez y a la separación estructural entre liquidez, deuda y patrimonio.

## Arquitectura del sistema

El sistema se compone de:

1. Definición conceptual y reglas financieras
2. Modelado relacional en PostgreSQL
3. Integración y modelo semántico en Power BI
4. Implementación de KPIs
5. Diseño funcional y UX/UI
6. Iteración y evolución controlada

## Estructura modular

- Módulo 0 – Conceptos y reglas
- Módulo 1 – Stack tecnológico
- Módulo 2 – Modelado SQL
- Módulo 3 – Integración Power BI
- Módulo 4 – Modelo semántico
- Módulo 5 – Sketches funcionales
- Módulo 6 – KPIs y visualización
- Módulo 7 – Iteración y mantenimiento
- Módulo 8 – UI/UX final
