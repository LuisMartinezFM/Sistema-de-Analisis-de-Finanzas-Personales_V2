# Módulo 4 – Modelo semántico en Power BI

## Objetivo

Construir una capa de interpretación coherente que respete las reglas definidas en SQL y permita análisis confiables.

## Decisiones clave

- Relaciones explícitas uno a muchos.
- Dirección de filtros controlada.
- Uso limitado de relaciones bidireccionales.
- Clasificación de movimientos neutros.
- Los valores (En blanco) se analizan, no se ocultan.

## Principio estructural

El modelo semántico interpreta los datos, no corrige errores del modelo relacional.
